#  Mintlify Landing Page Clone (HTML + CSS)

A **desktop-first clone** inspired by the Mintlify documentation website, built using **only HTML and CSS**.

This project focuses on:

- Layout accuracy  
- Content structure  
- Visual hierarchy  
- Documentation-style UI  
- Glassmorphism cards  
- Dark theme design  

No JavaScript. No frameworks.

---

##  Live Demo
https://mintlifyprject.netlify.app

- HTML5  
- CSS3 (Flexbox + Grid)  
- System UI Fonts
